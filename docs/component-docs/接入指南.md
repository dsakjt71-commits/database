# 接入指南

## NuGet 包

通用包:

```xml
<PackageReference Include="Tyouqu.Database.Abstractions" Version="1.0.0" />
<PackageReference Include="Tyouqu.Database.Dapper" Version="1.0.0" />
```

统一入口包:

```xml
<PackageReference Include="Tyouqu.Database" Version="1.0.0" />
```

按数据库选择一个 Provider:

```xml
<PackageReference Include="Tyouqu.Database.SqlServer" Version="1.0.0" />
<PackageReference Include="Tyouqu.Database.MySql" Version="1.0.0" />
<PackageReference Include="Tyouqu.Database.PostgreSql" Version="1.0.0" />
<PackageReference Include="Tyouqu.Database.Sqlite" Version="1.0.0" />
```

## 注册服务

统一入口（根据 `DatabaseOptions.Provider` 自动选择数据库实现）:

```csharp
using Tyouqu.Database.Abstractions;
using Tyouqu.Database.DependencyInjection;

builder.Services.AddTyouquDatabase(options =>
{
    options.Provider = DatabaseProvider.SqlServer;
    options.ConnectionString = builder.Configuration.GetConnectionString("Default")!;
    options.CommandTimeoutSeconds = 30;
    options.SqlTemplates.RootPath = "sql";
});
```

SQL Server:

```csharp
using Tyouqu.Database.Abstractions;
using Tyouqu.Database.SqlServer.DependencyInjection;

```

## SQL 目录

```text
sql
  common
  sqlserver
  mysql
  postgresql
  sqlite
```

SQL ID 等于文件名去掉 `.sql`:

```text
user.getById.sql -> user.getById
file.create.sql  -> file.create
```

该规则对 SQL Server、MySQL、PostgreSQL、SQLite 完全一致。示例：

```text
sql/sqlserver/auth/auth.app.getById.sql
sql/mysql/auth/auth.app.getById.sql
sql/postgresql/auth/auth.app.getById.sql
sql/sqlite/auth/auth.app.getById.sql
```

调用时统一使用：

```csharp
await db.QuerySingleOrDefaultByIdAsync<App>(
    "auth.app.getById",
    new { AppId = 1 });
```
