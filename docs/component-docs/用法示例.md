# 用法示例

## 查询单条

```csharp
var app = await db.QuerySingleOrDefaultByIdAsync<App>(
    "auth.app.getById",
    new { AppId = 1 });
```

SQL 文件：

```text
sql/sqlserver/auth/auth.app.getById.sql
```

```sql
select
    app_id as AppId,
    app_name as AppName,
    app_key as AppKey,
    status as Status,
    created_at as CreatedAt
from auth_apps
where app_id = @AppId;
```

## 查询列表

```csharp
var apps = await db.QueryByIdAsync<App>(
    "auth.app.list",
    new { Keyword = "%test%" });
```

SQL 文件：

```text
sql/common/auth/auth.app.list.sql
```

```sql
select
    app_id as AppId,
    app_name as AppName,
    app_key as AppKey,
    status as Status,
    created_at as CreatedAt
from auth_apps
where (@Keyword is null or app_name like @Keyword or app_key like @Keyword)
order by created_at desc;
```

## 分页

```csharp
var result = await db.QueryPagedByIdAsync<App>(
    "auth.app.list",
    new { Keyword = keyword },
    new PageRequest(1, 20));
```

分页可以复用列表 SQL。组件会自动生成 count SQL，并通过当前数据库方言追加分页语句。


SQL 文件：

```text
sql/common/auth/auth.app.delete.sql
```


## 新增和更新

```csharp
await db.ExecuteByIdAsync(
    "auth.app.create",
    new
    {
        AppName = "Demo",
        AppKey = "demo",
        Status = 1
    });

await db.ExecuteByIdAsync(
    "auth.app.update",
    new
    {
        AppId = 1,
        AppName = "Demo App",
        Status = 1
    });
```

新增 SQL：

```text
sql/common/auth/auth.app.create.sql
```

```sql
insert into auth_apps (
    app_name,
    app_key,
    status,
    created_at
) values (
    @AppName,
    @AppKey,
    @Status,
    current_timestamp
);
```

更新 SQL：

```text
sql/common/auth/auth.app.update.sql
```

```sql
update auth_apps
set app_name = @AppName,
    status = @Status,
    updated_at = current_timestamp
where app_id = @AppId;
```

## 多结果集

```csharp
var result = await db.QueryMultipleByIdAsync<App, AppStats>(
    "auth.app.getDetailWithStats",
    new { AppId = 1 });

var app = result.First.SingleOrDefault();
var stats = result.Second.SingleOrDefault();
```

SQL 文件：

```sql
select *
from Apps
where app_id = @AppId;

select
    count(1) as TotalApps,
    sum(case when status = 1 then 1 else 0 end) as EnabledApps
from Apps;
```

## 开启执行日志

.NET：

```csharp
builder.Services.AddTyouquDatabase(options =>
{
    options.SqlLogging.Enabled = true;
    options.SqlLogging.LogSql = true;
    options.SqlLogging.LogParameters = false;
    options.SqlLogging.SlowSqlThresholdMs = 500;
});
```

Java：

```java
DatabaseOptions options = DatabaseOptions.builder()
    .provider(DatabaseProvider.MYSQL)
    .enableSensitiveLogging(false)
    .sqlLogging(new SqlExecutionLogOptions(true, true, false, false, 500))
    .build();

JdbcDbExecutor db = new JdbcDbExecutor(
    dataSource,
    options,
    List.of(new FullTableOperationInterceptor(options.safety())),
    List.of(new ConsoleSqlExecutionLogger())
);
```

Python：

```python
logs = []
options = DatabaseOptions(
    provider=DatabaseProvider.SQLITE,
    connection_factory=lambda: sqlite3.connect("app.db"),
    sql_logging=SqlExecutionLogOptions(enabled=True, log_sql=True),
)

db = SqliteDbExecutor(
    options.connection_factory,
    options,
    execution_loggers=[logs.append],
)
```

执行日志可用于查询慢 SQL、失败 SQL 和异常 SQL。参数默认不会输出，生产环境谨慎开启敏感参数日志。

## 全表更新和删除阻断

以下 SQL 会被默认阻断：

```sql
delete from auth_apps
```

```sql
update auth_apps
set status = 0
```

应改为带条件的 SQL：

```sql
delete from auth_apps
where app_id = @AppId
```
