# 测试说明

## 单元测试

位置：

```text
tests/Tyouqu.Database.UnitTests/TyouquDatabase
```

覆盖：

- common/provider SQL 加载
- provider 覆盖 common
- SQL ID 缺失
- 禁止旧 `-- @id` 格式
- 空 SQL 文件报错
- 同一作用域重复 SQL ID 报错
- SQL Server 分页方言和标识符转义
- MySQL 分页方言和标识符转义
- PostgreSQL 分页方言和标识符转义
- SQLite 分页方言和标识符转义
- SQLite Provider 真实执行、分页、多结果集

运行：

```powershell
dotnet test tests\AuthCenter.UnitTests\AuthCenter.UnitTests.csproj --filter FullyQualifiedName~TyouquDatabase
```
