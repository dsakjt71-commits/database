# Tyouqu.Database 功能文档

## 功能定位

`Tyouqu.Database` 是一个跨项目复用的数据库基础组件。它不负责业务规则，只提供稳定、统一、可扩展的数据库访问能力。

## 当前功能

当前目标是提供一套可在多个业务服务复用的数据库访问组件，减少 SQL Server 存储过程绑定，避免 SQL 写死在代码中，并支持 SQL Server、MySQL、PostgreSQL、SQLite 四个 Provider。

### 1. 数据库连接管理

提供统一连接工厂：

```text
IDbConnectionFactory
```

能力：

- 创建打开状态的数据库连接。
- 屏蔽不同数据库驱动差异。
- 统一连接字符串配置。
- 支持连接池配置。

### 2. 参数化 SQL 执行

提供：

```text
IDbExecutor
```

能力：

- 执行非查询 SQL。
- 查询列表。
- 查询单条。
- 查询标量值。
- 支持匿名对象参数。
- 支持 CancellationToken。

### 3. 外置 SQL 模板

提供：

```text
ISqlTemplateStore
ISqlTemplateExecutor
```

能力：

- 从 SQL 文件加载 SQL 模板。
- 通过 SQL ID 获取 SQL。
- 支持 common SQL 和 Provider 专用 SQL。
- 支持启动加载。
- 支持手动重载。

### 4. Provider 支持

当前已支持：

```text
SQL Server
MySQL
PostgreSQL
SQLite
```

共同能力：

- Provider 专用连接创建。
- Provider 专用分页方言。
- Provider 专用标识符转义。
- Provider 专用 SQL 模板目录。
- 统一入口 `AddTyouquDatabase` 可根据 `DatabaseOptions.Provider` 自动选择实现。

### 5. 事务管理

提供：

```text
IUnitOfWork
IDbTransactionScope
```

能力：

- 开启事务。
- 提交事务。
- 回滚事务。
- 同一事务内复用连接。

### 6. 分页支持

提供统一分页模型：

```text
PageRequest
PagedResult<T>
```

能力：

- 自动计算 offset。
- 自动生成分页 SQL。
- 返回列表和总数。

### 7. 日志和异常

能力：

- 记录 SQL ID。
- 记录耗时。
- 记录影响行数。
- 慢 SQL 日志。
- 统一包装数据库异常。
- 默认不记录敏感参数。

## 后续功能

### 1. SQL 模板校验工具

提供命令行或测试辅助能力：

```text
检查 SQL ID 是否重复
检查 SQL ID 是否缺失
检查 SQL 文件是否能解析
检查不同 Provider 是否存在必需 SQL
```

### 2. 手动 Reload 管理接口

业务服务可选择暴露内部接口：

```http
POST /internal/database/sql-templates/reload
```

接口调用后：

- 重新扫描 SQL 文件。
- 解析成功后整体替换旧模板。
- 解析失败则继续使用旧模板。

## 规划功能

### 1. 通用 Repository

提供可选通用 Repository：

```text
IRepository<TEntity, TKey>
```

能力：

- GetById
- Insert
- Update
- Delete
- SoftDelete
- Exists

该功能可选，不强迫业务系统使用。

### 2. 简单 SQL 模板条件语法

支持有限动态条件：

```sql
{{#if Keyword}}
and title like @Keyword
{{/if}}
```

第一版不建议实现，避免复杂度过高。

### 3. 读写分离

支持：

- 主库写。
- 从库读。
- 强制主库读。
- 读库故障降级。

### 4. 多租户数据库

支持：

- 按租户选择连接字符串。
- 按租户选择 schema。
- 按租户选择 SQL 模板。

## 不包含的功能

第一版不包含：

- ORM 级别变更追踪。
- 自动生成复杂 SQL。
- 数据库迁移系统。
- 复杂模板引擎。
- 图形化 SQL 管理后台。
- 业务实体规则。

如果后续需要 EF Core，可以单独增加：

```text
Tyouqu.Database.EfCore
```

## 与业务系统边界

组件负责：

```text
连接
执行
事务
分页
SQL 模板
Provider 适配
日志
异常
```

业务系统负责：

```text
表结构设计
SQL 内容
业务 Repository
业务校验
权限判断
数据返回模型
```
